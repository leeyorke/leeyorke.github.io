# Soul


- 宜早不宜迟

- 要有目标

- 每年与家人许一个愿望，并完成它

- 能用大于完美（在东西做出来之前不要追求完美）

- 可以计划的事情务必先做计划，不能计划的事情先做，发现问题后再解决问题。

- 允许悲观与沮丧，但只能是暂时的，不能一直跌倒不爬起来。

- 以德报德，以怨报怨。

- 能跟得上时代脚步的人，会比跟不上时代脚步的人多一份选择。【比如70后会电脑的和不会电脑的】

- 赞誉要公开、慷慨、及时；责备要私下、具体、建设性。

- 问题问得好，事情就解决了一半。- 约翰·杜威。

- 吃好、睡好、养成良好的生活习惯，会解决80%的问题。

- 我能为他人解决什么独特的痛点？我真正的优势和价值在哪里？

- 对我而言，幸福的瞬间是由什么构成的？”“我内心深处的恐惧和渴望是什么？”“我如何与不完美（包括自己的和他人的）共处？

- 什么事能让我忘记时间？”“如果我不必担心失败，我最想尝试的是什么？”“我这一生，想要为这个世界留下一点什么不同？

- 脚踏实地地写出一些有价值的作品

- 根据力的相互作用，你荒废时间，时间也荒废了你。

- 你正在寻找的东西也在寻找你。

- 我宁愿死，也要去做一件我觉得很伟大很了不起的事，而不是安全舒适地过自己讨厌的生活。

- 你越逃避恐惧，恐惧越强大，但你越是直面恐惧，恐惧越会像幻象那样消失不见。

- 你当下拥有的就是最好的。
