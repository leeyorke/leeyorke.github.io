yaml 中多行字符串的各种写法。

原始yaml

1
2
3
4
5

paragraph:
  Blank lines denote
  paragraph breaks
alias:
  bar: baz

转成json

1
2
3
4
5
6

{
  "paragraph": "Blank lines denote paragraph breaks",
  "alias": {
    "bar": "baz"
  }
}

|

作用：| 会给每一行末尾加上 \n 字符串。

示例

1
2
3
4
5

paragraph: |
  Blank lines denote
  paragraph breaks  
alias:
  bar: baz

转成json

1
2
3
4
5
6

{
  "paragraph": "Blank lines denote\nparagraph breaks\n",
  "alias": {
    "bar": "baz"
  }
}

|-

作用：|- 在 | 的基础上会将最后一行末尾的 \n 去掉。

示例

1
2
3
4
5

paragraph: |-
  Blank lines denote
  paragraph breaks  
alias:
  bar: baz

转成json

1
2
3
4
5
6

{
  "paragraph": "Blank lines denote\nparagraph breaks",
  "alias": {
    "bar": "baz"
  }
}

>

作用：> 会给最后一行末尾加上 \n 字符串。

示例

1
2
3
4
5

paragraph: >
  Blank lines denote
  paragraph breaks  
alias:
  bar: baz

转成json

1
2
3
4
5
6

{
  "paragraph": "Blank lines denote paragraph breaks\n",
  "alias": {
    "bar": "baz"
  }
}

>-

作用：>- 在 > 基础上会将最后一行末尾 \n 去掉。

示例

1
2
3
4
5

paragraph: >-
  Blank lines denote
  paragraph breaks  
alias:
  bar: baz

转成json

1
2
3
4
5
6

{
  "paragraph": "Blank lines denote paragraph breaks",
  "alias": {
    "bar": "baz"
  }
}

总结

操作符 含义 等效于 备注
| 给每一行的末尾加上\n 无 无
|+ 给每一行的末尾加上\n | 不常用
|- 去掉最后一行末尾的\n 无 常用
> 给最后一行的末尾加上\n 无 常用
>+ 给最后一行的末尾加上\n > 不常用
>- 去掉最后一行的末尾加上\n 什么符号也不加 不常用

常用：

  • |
  • >
  • |-
